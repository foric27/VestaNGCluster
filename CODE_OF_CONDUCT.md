# Code of Conduct

## Наши обязательства

Мы стремимся сделать участие в проекте приятным и продуктивным для всех.

## Стандарты поведения

Примеры позитивного поведения:
- Использование приветливого и инклюзивного языка
- Уважение к различным точкам зрения
- Конструктивная критика
- Фокус на том, что лучше для сообщества

Примеры неприемлемого поведения:
- Троллинг, оскорбления, персональные attacks
- Публикация личной информации
- Непристойное поведение

## Область применения

Этот кодекс применяется во всех пространствах проекта.

## Применение

О случаях неприемлемого поведения можно сообщить через GitHub Issues или приватно.
